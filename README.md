# 🛒 Online Groceries Shopping App (SwiftUI + Node.js)

A modern grocery shopping application built using **SwiftUI** for iOS and **Node.js + MySQL** for the backend. The app provides a seamless shopping experience with product browsing, category management, cart functionality, user authentication, and order management.

## 🚀 Features

### User Features

* User Registration & Login
* Secure Authentication
* Browse Grocery Categories
* View Product Details
* Add Products to Cart
* Update Cart Quantity
* Wishlist Management
* Order Placement
* Order History
* Profile Management

### Admin Features

* Category Management
* Brand Management
* Product Management
* Order Management
* Customer Management

## 🛠️ Tech Stack

### Frontend

* SwiftUI
* MVVM Architecture
* URLSession Networking
* Async/Await
* Custom UI Components

### Backend

* Node.js
* Express.js
* MySQL
* REST APIs
* JWT Authentication

### Database

* MySQL

## ⚙️ Backend Setup

```bash
# Install dependencies
npm install

# Import database
mysql -u root groceries < groceries.sql

# Run server
node app.js
```

Server runs on:

```text
http://localhost:3001
```
